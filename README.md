# Ludomicile

Site web professionnel pour LUDO MICILE (Société LANGE), entreprise de maçonnerie à Tergnier (02).

## Description

Ce site présente les services de maçonnerie générale, rénovation et construction offerts par LUDO MICILE. Il met en avant l'expertise, la qualité et le savoir-faire traditionnel de l'entreprise.

## Fonctionnalités

- Présentation des services (maçonnerie générale, rénovation, construction)
- Portfolio des réalisations
- Informations de contact et devis gratuit
- Design responsive et moderne
- Intégration des réseaux sociaux (Facebook)

## Technologies utilisées

- HTML5
- CSS3 (avec animations et design responsive)
- JavaScript (pour les interactions utilisateur)
- Font Awesome pour les icônes
- Google Fonts pour la typographie

## Installation et utilisation

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/momort02/ludomicile.git
   ```

2. Ouvrez le fichier `index.html` dans votre navigateur web.

Le site est également hébergé sur GitHub Pages : [https://momort02.github.io/ludomicile](https://momort02.github.io/ludomicile)
